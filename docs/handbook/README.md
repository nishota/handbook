# Handbook

## ハンドブックについて

本ハンドブックについての目的、利用方法、更新方法、レビュー方法についてまとめていく。

本ハンドブックは[ハンドブックファースト](./handbook-first)の理念のもと、グループ運営に関する情報のSSoT（唯一の信頼できる情報源）として運用する。
すべての記述は[下書き](./handbook-first/handbook-is-a-draft)であり、[ハンドブックレビュー](./review)を経て継続的に改善していく。
