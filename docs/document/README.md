# Document

## ドキュメントについて

ドキュメントの作成・管理に関するルール、フォーマット、レビュー方法などについてまとめていく。

ハンドブックと同様に、ドキュメントはグループの知見を蓄積・共有するための重要な資産である。誰もが参照しやすく、かつ最新の状態を維持できるようルールを整理する。

本セクションは[ハンドブックファースト](../handbook/handbook-first)の理念に従い、ドキュメント運用に関するルールのSSoTとして運用する。

## 扱う内容

- ドキュメント作成の原則
- 管理場所と分類
- フォーマットと書き方のガイドライン
- レビューと更新
